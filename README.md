A modern weather application built with Node.js, Express, and EJS that allows users to:
- Search weather by city  
- View temperature, humidity, wind, sunrise & sunset  
- See a 5-day forecast  
- Use geolocation to get local weather  
- Dynamic backgrounds based on weather conditions  


- Node.js  
- Express  
- EJS  
- OpenWeather API  
- HTML, CSS, JavaScript
